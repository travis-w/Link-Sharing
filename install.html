Link Sharing

By: Travis Weidenbenner
This extension allows you to easily share links with others via Google Chrome. I hacked this together one weekend because I thought it could be useful for me and some friends. So now I am tidying up the code and releasing it as Open Source.

  Accounts Needed

In order to finish setting up this extension you will need the following:
1) A CloudMine account (stores your links)
2) A BitPixels account (displays site thumbnails)
(If someone else already has the codes for you or you are updating, then you can ignore this)
